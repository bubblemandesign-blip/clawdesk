⚠️ Windows shows a security warning?

This is normal for apps without a code signing certificate ($200+/year).

1. Click "More info"
2. Click "Run anyway"

ClawDesk is open source — you can inspect every line of code.

⚠️ Mac says the app cannot be opened?

Option A: Right-click ClawDesk → Open → Click Open

Option B: If you see "app is damaged", open Terminal and run:

xattr -cr /Applications/ClawDesk.app

🐧 Linux setup

chmod +x ClawDesk-Linux.AppImage

./ClawDesk-Linux.AppImage

If it fails, try: ./ClawDesk-Linux.AppImage --no-sandbox
